# Design Question Bank (Design) — starter template

> Used by lode-brief. Read and written in place (`skills/lode-brief/references/question-bank-design.md`) — self-evolution only edits it here, no `.lode/` copy.
> Force "feel" into decisions: multiple-choice first, anti-reference mandatory, rejected → swap in a new batch.
> Self-evolution only adds/changes/deletes here, each mapping to one real failure.

| Axis | Question (multiple-choice + give a recommendation) | ✅ Acceptable | ❌ Push back |
|---|---|---|---|
| Design direction | ①editorial ②neo-brutalism ③glassmorphism ④Swiss… (suggest X) | Clearly pick one | Empty phrases like "clean minimal" |
| Combo reference | "A's structure + B's vibe" — offer 2–3 to pick | Pick one + say what's alike | Can't name a reference |
| **Anti-reference** | **What must never appear? Which products' feel do you hate?** | List 1–3 counter-examples | Left blank (mandatory) |
| Theme color | Dark / light / both? | Pick one | Vague |
| Palette | Offer one semantic palette (rejected → swap batch) | User approves a set | Stuck on "too AI" without swapping |
| Information density | Compact / medium / loose? | Pick one | —— |
| Typography | 2–3 pairings to pick from | Pick one | —— |
| Corner-radius hierarchy | Sharp tool / soft modern / large consumer radius? | Pick one | —— |
| Key states | hover/focus/active/empty/loading/error — how presented? | All accounted for | States missing |
| Copy tone | Tool-like / agent-anthropomorphic / hybrid? | Pick one (align to Spec) | —— |

> Add project-specific axes; the anti-reference row **must be asked**.
